# Kyusu(急須)

## これは何?
テキスト整形ツールです。
自然言語処理技術を利用する前に必要な最低限の処理を実装しています。
基本的にルールベースです。文字コードはUTF-8を前提にしています。
この手のツールは凝り始めるといくらでも深掘りできますが、
現状ざっくり割り切った実装になっています。

## 付属コマンド
* sensplit
文切りツールです。段落などの文に区切られていないテキストを文ごとに切り分けます。
単純なルールベースで作成されています。

括弧内は文の一部と見なして切り分けません。
例：以下の文は1文となります。
彼は「カレーを食べたい。」と言った。

句点（。）を含む固有名詞などの特殊処理は一切行いません。
顔文字についても未対応です。

* preprocess
3文字以下の文、または1000文字以上の文を削除します。
全角スペースを半角スペースに変換します。
文の前後のスペースを削除します。

* postprocess
形態素解析後の事後処理です。
形態素解析によって細かく切られすぎてしまった単語を再結合します。
アルファベット・数字・ハイフンの連続を結合します。
ハイフンの場合は前後のカタカナも結合します。

## 利用例
$ sensplit < textfile | preprocess | mecab -Owakati | postprocess

## 著作権について
本プログラムではUnidic 2.1.2 ( http://www.ninjal.ac.jp/corpus_center/unidic/ )
より句点類と括弧類を抽出させていただきました。これらのデータは国立国語研究所が保持し、オープンソースライセンス( GPL/LGPL/BSD ライセンス )のもとで提供されているものです。抽出されたデータはdataディレクトリに収められています。
